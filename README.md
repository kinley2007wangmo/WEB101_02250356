 # Profile Card React Application

## Project Overview

This project is a simple profile card application built using React and Vite.
It displays a user profile with a profile picture, name, role, skills, and an interactive Like button.

## Features

* Display profile picture
* Show name and role
* List user skills
* Interactive Like button using React useState
* Component-based React structure

## Technologies Used

* React
* Vite
* JavaScript
* CSS
* HTML

## How to Run the Project

1. Install dependencies
   npm install

2. Start the development server
   npm run dev

3. Open the browser and go to
   http://localhost:5173
